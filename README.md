## Technologies
Project is created with:
* Java 11
* JUnit 4.13.2
* maven 3.9.2
* Mockito 5.2.0
* Jacoco 0.8.7

## Setup
To run this project:

```
$ cd Sprint_4
$ mvn compile 
$ mvn clean test
```
